Repository 생성

Repository 생성

git init

Repository를 생성할 폴더에 GitBash, Cmd, PowerShell, Terminal과 같은 프로그램을 켜준뒤 다음과 같은 명령어를 입력한다.

그러면 그 폴더에 .git이라는 숨겨진 폴더가 생성이 된다.
이 폴더에는 Git의 Repository를 구성하기 위한 기본적인 파일들이 들어있다.
